# 🛡️ 职场边界防护罩 (Workplace Boundary Shield)

> 把"关你屁事"翻译成对方听不出来的商务普通话。

一个专为职场越界行为设计的沟通 AI Skill。当同事打探私事、领导爹味建议、或任何人越权干涉你的计划时，一键生成**三档丝滑闭嘴话术**。

---

## ✨ 核心功能

| 档位 | 适用对象 | 核心策略 |
|------|----------|----------|
| **Level 1 礼貌闭环** | 半熟人 / 平级 | 感谢 + 顺其自然 → 封死追问空间 |
| **Level 2 体贴式驱逐** | 长辈 / 老板 / 热心同事 | 表面为对方省时间，实则拒绝沟通 |
| **Level 3 高级 Mean 语** | 居高临下 / 打探越界者 | 优雅地说"关你屁事"，对方还挑不出毛病 |

---

## 🚀 使用方法

在支持 Skill 的 AI IDE（如 Gemini/Claude）中加载后，直接输入对方说的话：

```
"听说你最近在看新机会？准备去哪儿啊？"
```

Skill 会自动识别场景类型，并输出三档回复 + 潜台词说明。

## 📐 MYOB Matrix 核心逻辑

1. **礼貌闭环 (Safe Close)** — 让对方以为你听进去了，实则没有
2. **体贴式驱逐 (Courteous Dismissal)** — 让对方以为你在为他省事，实则在赶人
3. **逻辑重塑 (Reframing)** — 把越界变成"因人而异"的哲学问题，让对方无从追问
